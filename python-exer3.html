1. 함수 (Function)

함수는 특정 기능을 수행하는 코드의 묶음입니다. 코드를 재사용하고 효율적으로 관리할 수 있게 해줍니다.

1-1. 함수 정의 및 호출

함수를 정의하고 호출하는 가장 기본적인 형태입니다. 아래 입력 필드에 이름을 넣고 '호출하기' 버튼을 눌러보세요.

def greet(name):
    print(f"안녕하세요, {name}님!")

greet("홍길동")

함수가 어떤 값을 계산하여 돌려주는 경우입니다. 두 숫자를 입력하고 '더하기' 버튼을 눌러 결과를 확인하세요.

def add(x, y):

1-3. 기본 매개변수

매개변수에 기본값을 지정해, 값을 전달하지 않아도 오류가 발생하지 않게 합니다. 이름과 나이(선택)를 입력해보세요.

def show_info(name, age=0):
    print(f"이름: {name}, 나이: {age}")

show_info("영희")
show_info("민수", 25)

2. 모듈 (Module)

모듈은 파이썬 코드가 저장된 파일입니다. 다른 사람이 만든 유용한 기능을 가져와 사용할 수 있습니다.

2-1. 모듈 가져오기

다양한 모듈을 가져와 내장된 기능을 사용할 수 있습니다. 아래 버튼을 눌러 각 모듈의 기능을 실행해보세요.

# 'math' 모듈을 통째로 가져와서 사용합니다.
import math
print(f"파이 값: {math.pi}")

# 'random' 모듈에 별칭을 부여하여 사용합니다.
import random as r
print(f"랜덤 정수 (1 ~ 10): {r.randint(1, 10)}")

# 'datetime' 모듈에서 'datetime' 클래스만 가져옵니다.
from datetime import datetime
print(f"현재 시간: {datetime.now()}")

3. 파일 입출력

파이썬으로 파일을 읽거나 쓸 수 있습니다. 이 예제에서는 브라우저 내에서 파일 입출력을 시뮬레이션합니다.

3-1. 파일 쓰기

텍스트 에어리어에 내용을 입력하고 '파일 쓰기' 버튼을 누르면 가상의 'my_file.txt'에 내용이 저장됩니다.

with open("my_file.txt", "w", encoding="utf-8") as f:
    f.write("안녕하세요, 파일 입출력입니다.\\n두 번째 줄입니다.")

print("my_file.txt 파일이 생성되었습니다.")

3-2. 파일 읽기

'파일 읽기' 버튼을 누르면 위에서 저장한 가상 파일의 내용을 불러옵니다.

with open("my_file.txt", "r", encoding="utf-8") as f:
    content = f.read()

print(content)

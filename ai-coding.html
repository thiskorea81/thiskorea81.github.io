AI와 함께 생각하며 코딩하기

AI를 단순한 코드 생성기가 아닌, 당신의 생각을 현실로 만들어주는 '생각의 파트너'로 활용하는 방법을 배워보세요. 이 가이드는 체계적인 3단계 접근법을 통해 여러분이 더 나은 개발자로 성장하도록 돕습니다.

🤔 왜 그냥 "코딩해줘"라고 하면 안 될까요?

🖥️

모호한 화면 구성

AI는 당신이 원하는 결과물이 웹사이트인지, 앱인지, 아니면 단순한 텍스트 기반 프로그램인지 알 수 없습니다.

🧩

불명확한 핵심 기능

AI는 '추가', '삭제', '수정' 중 어떤 기능이 필요한지, 그리고 어떻게 동작해야 하는지 스스로 판단할 수 없습니다.

💾

데이터 저장 방식

데이터를 임시로만 저장할지, 파일로 남길지, 데이터베이스에 기록할지 명확한 지시가 필요합니다.

부정확한 질문은 원하지 않는 결과로 이어집니다. 원하는 프로그램을 정확히 얻으려면 컴퓨터처럼 생각하는 연습이 필요합니다!

Step 1

문제 정의 (WHAT)

'어떻게'가 아닌 '무엇을' 만들지에 집중하여 아이디어를 구체화하는 단계입니다.

예시: "오늘의 할 일 목록(To-Do List) 관리 프로그램"

🎯

주요 목표

사용자가 텍스트(콘솔) 환경에서 할 일을 추가, 조회, 완료 처리하는 프로그램을 만듭니다.

📝

핵심 기능

  • 추가: 새로운 할 일을 목록에 더하기
  • 보기: 등록된 모든 할 일을 번호와 함께 표시
  • 완료: 특정 할 일을 "(완료)" 상태로 변경
  • 종료: 프로그램 끝내기
⌨️

입력(Input) / 출력(Output)

입력: 사용자의 키보드 입력 (메뉴 번호, 할 일 내용)

출력: 화면에 표시되는 텍스트 (메뉴, 목록, 안내 메시지)

⛓️

제약 조건

데이터는 프로그램 실행 중에만 유지됩니다 (파일 저장 X).

Step 2

알고리즘 구상 (HOW)

코딩 문법 대신 일상 언어로 프로그램의 '행동 순서'를 계획하는 단계입니다.

전체 프로그램의 기본 흐름

1. 프로그램 시작 메시지 표시
↓

사용자가 '종료'를 선택할 때까지 무한 반복

a. 메뉴 표시 (추가, 목록, 완료, 종료)
b. 사용자에게 메뉴 번호 입력받기
c. 입력 번호에 따라 해당 기능 실행
d. '4' 선택 시, 반복 중단
↓
2. 프로그램 종료 메시지 표시

세부 기능별 절차

▶ '할 일 추가' 절차
  1. "추가할 할 일을 입력하세요:" 라고 묻는다.
  2. 입력받은 내용을 '할 일 목록'에 저장한다.
  3. "추가되었습니다." 메시지를 보여준다.
▶ '목록 보기' 절차
  1. 만약 목록이 비어있으면 "할 일이 없습니다." 표시
  2. 그렇지 않으면, 목록의 각 항목 앞에 번호를 붙여 순서대로 표시
▶ '할 일 완료 처리' 절차
  1. 현재 할 일 목록을 전부 보여준다.
  2. "완료할 할 일의 번호를 입력하세요:" 라고 묻는다.
  3. 입력받은 번호에 해당하는 할 일을 찾는다.
  4. 찾은 할 일 내용 앞에 "(완료)" 텍스트를 붙인다.
Step 3

코딩 요청 (CODE)

정의와 구상을 합쳐, 원하는 언어를 명시하여 AI에게 코드를 요청하는 단계입니다.

🤖 AI에게 보낼 프롬프트(요청) 예시

오늘의 할 일을 관리하는 간단한 파이썬(Python) 콘솔 프로그램을 만들고 싶어.


[주요 기능]
- 새로운 할 일을 목록에 추가
- 현재 할 일 전체 목록을 번호와 함께 표시
- 번호를 선택해서 특정 할 일을 완료 상태로 변경 (완료된 항목은 앞에 "(완료)" 표시)
- 프로그램 종료


[동작 방식]
- 프로그램이 시작되면 사용자에게 [1. 추가, 2. 목록 보기, 3. 완료 처리, 4. 종료] 메뉴를 보여줘.
- 사용자가 메뉴를 선택하면 해당 기능을 실행하고, 다시 메뉴를 보여주는 과정을 반복해야 해.
- 사용자가 '4'를 입력하면 프로그램을 종료하면 돼.


위 요구사항에 맞춰 파이썬 코드를 작성해 줘.

🐍 AI가 생성한 파이썬 코드 결과

def main():
    todo_list = []

    while True:
        print("\n--- 오늘의 할 일 목록 ---")
        print("1. 할 일 추가")
        print("2. 목록 보기")
        print("3. 완료 처리")
        print("4. 종료")
        
        choice = input("메뉴를 선택하세요: ")

        if choice == '1':
            task = input("추가할 할 일을 입력하세요: ")
            todo_list.append(task)
            print("할 일이 추가되었습니다.")
        
        elif choice == '2':
            if not todo_list:
                print("할 일이 없습니다.")
            else:
                print("\n[전체 할 일 목록]")
                for i, task in enumerate(todo_list):
                    print(f"{i+1}. {task}")
        
        elif choice == '3':
            # (이하 완료 처리 로직)
            try:
                task_num = int(input("완료할 번호: "))
                if 1 <= task_num <= len(todo_list):
                    if not todo_list[task_num-1].startswith("(완료)"):
                        todo_list[task_num-1] = "(완료) " + todo_list[task_num-1]
                        print("완료 처리되었습니다.")
                else:
                    print("잘못된 번호입니다.")
            except ValueError:
                print("숫자를 입력해주세요.")

        elif choice == '4':
            print("프로그램을 종료합니다.")
            break
        
        else:
            print("잘못된 입력입니다.")

if __name__ == "__main__":
    main()

✅ 정리 및 복습

코딩은 문법 암기가 아닌, 문제를 해결하는 사고의 과정입니다.

1

정의하기 (What)

무엇을 만들지 명확히 합니다.

2

구상하기 (How)

어떻게 동작할지 순서를 정합니다.

3

요청하기 (Code)

정의와 구상을 합쳐 코드를 요청합니다.

이제 AI와 함께 여러분만의 프로젝트를 시작해보세요!

문자열 포매팅 (String Formatting)

변수 값을 문자열에 삽입하는 다양한 방법을 알아봅니다. '실행' 버튼을 눌러 결과를 확인하세요.

3. 조건문 (If-elif-else)

조건문은 특정 조건이 참(True)일 때만 코드 블록을 실행합니다. 점수를 입력하고 '결과 확인' 버튼을 눌러 어떤 학점 메시지가 출력되는지, 그리고 코드의 어느 부분이 실행되는지(하이라이트) 확인해 보세요.

score = 85 if score >= 90: print("A 학점입니다.") elif score >= 80: print("B 학점입니다.") elif score >= 70: print("C 학점입니다.") else: print("F 학점입니다.")

4. 반복문 (For, While)

반복문은 특정 작업을 여러 번 수행할 때 사용됩니다. '반복 실행' 버튼을 눌러 코드가 한 줄씩 실행되며 결과가 출력되는 과정을 시각적으로 확인해 보세요.

For 반복문 (리스트 순회)

fruits = ["사과", "바나나", "체리"] for fruit in fruits: print(f"'{fruit}' 맛있어요!")

While 반복문

count = 0 while count < 3: print(f"현재 카운트: {count}") count += 1

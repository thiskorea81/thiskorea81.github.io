🚀 HTML 첫걸음
소개 기본 구조 핵심 태그

홈페이지 제작, 레고 블록처럼 쉽고 재미있게!

HTML은 웹페이지의 뼈대를 만드는 언어입니다. 이 가이드는 홈페이지를 처음 만드는 분들을 위해 가장 필수적인 문법을 소개합니다. 각 섹션을 따라 직접 코드를 만져보며 HTML의 세계를 탐험해 보세요.

1. HTML 문서의 기본 구조

모든 HTML 문서는 아래와 같은 정해진 틀 안에서 만들어집니다. 각 항목을 클릭하여 코드에서 어떤 부분인지 확인해보세요.

<!DOCTYPE html>

이 문서가 HTML5 형식임을 브라우저에 알리는 선언입니다.

<html>

HTML 문서의 시작과 끝을 나타내며, 모든 내용은 이 태그 안에 있어야 합니다.

<head>

페이지 제목, 스타일 등 웹페이지에 직접 보이지 않는 정보를 담습니다.

<title>

웹 브라우저 탭에 표시될 페이지의 제목을 설정합니다.

<body>

웹페이지에 실제로 보이는 모든 내용(글자, 이미지 등)이 들어가는 부분입니다.

<!DOCTYPE html> <html> <head> <title>페이지 제목</title> </head> <body> <h1>안녕하세요!</h1> <p>홈페이지 내용이 들어갑니다.</p> </body> </html>

2. 꼭 알아야 할 기본 태그

HTML 태그를 이용해 내용을 꾸미고 구성할 수 있습니다. 아래 탭을 눌러 각 태그의 사용법을 익히고, 직접 코드를 수정하여 실시간으로 결과를 확인해보세요.

<h1> ~ <h6> 태그로 제목의 중요도를, <p> 태그로 문단을, <br> 태그로 줄바꿈을 표현합니다.

<strong>(굵게), <em>(기울임), <u>(밑줄) 태그로 텍스트를 강조할 수 있습니다.

<ul>(순서 없는 목록)과 <ol>(순서 있는 목록)으로 깔끔한 목록을 만들 수 있습니다. 각 항목은 <li> 태그를 사용합니다.

<a> 태그로 다른 페이지로 연결되는 링크를, <img> 태그로 이미지를 삽입할 수 있습니다.

1차시: 파이썬 기본 문법과 데이터 다루기

인공지능 실습에 필요한 데이터를 담고 조작하는 가장 기본적인 방법들을 배웁니다.

1. 변수 (Variables)

변수는 데이터를 담는 그릇입니다. 모델을 학습시킬 때 원인(독립 변수)과 결과(종속 변수)를 각각 다른 변수에 담아서 사용합니다.

# 실습 코드 미리보기: 데이터프레임(df3)에서 특정 열을 변수 X와 y에 저장합니다.
X = df3[['꽃잎 길이', '꽃잎 너비']] # 독립 변수 (원인 데이터)
y = df3['종류']                    # 종속 변수 (결과 데이터)

2. 리스트 (List)

리스트는 여러 개의 데이터를 대괄호 [ ] 로 묶어서 하나로 관리하는 방법입니다. AI가 새로운 꽃을 예측할 때, 그 꽃의 특징을 리스트로 전달합니다.

# 꽃잎 길이가 1.0cm, 꽃잎 너비가 1.5cm인 새로운 데이터 (2차원 리스트 형태)
new_data = [[1.0, 1.5]] 

# 모델이 이 새로운 데이터의 종류를 예측합니다.
print(model.predict(new_data))

3. 리스트에 추가하기 (append)

빈 리스트를 만들어두고 append() 명령어를 사용하면 데이터를 차례대로 추가할 수 있습니다. 실험 결과를 순서대로 기록할 때 유용합니다.

acc = [] # 정확도를 저장할 빈 리스트 생성

# 실험 결과가 나올 때마다 리스트 끝에 추가합니다.
acc.append(0.95) 
acc.append(0.98)

print(acc) # 출력 결과: [0.95, 0.98]

2차시: 제어문과 외부 모듈의 활용

반복되는 작업을 자동화하고, 전 세계 개발자들이 만들어 둔 강력한 AI 도구들을 불러오는 방법을 배웁니다.

1. for 반복문

K-최근접 이웃(KNN) 알고리즘에서 최적의 이웃 수(k)를 찾으려면 여러 번 실험을 반복해야 합니다. for문을 사용하면 이 과정을 자동화할 수 있습니다.

k_range = range(1, 20) # 1부터 19까지의 범위 지정

# k값을 1부터 19까지 바꾸어가며 아래 코드를 반복 실행합니다.
for k in k_range:
    model = KNeighborsClassifier(n_neighbors = k)
    model.fit(X_train, y_train)
    acc.append(model.score(X_test, y_test))

2. 모듈 불러오기 (import)

우리가 처음부터 인공지능을 다 만들 수는 없습니다. import를 사용하여 Scikit-learn(사이킷런) 같은 기계학습 라이브러리를 가져옵니다.

# 데이터 분석을 위한 pandas 모듈을 불러오고, 'pd'라는 별명으로 부릅니다.
import pandas as pd

# 기계학습 모듈(sklearn)에서 KNN 분류기 기능만 쏙 뽑아옵니다.
from sklearn.neighbors import KNeighborsClassifier
from sklearn.model_selection import train_test_split

3. 클래스(Class)와 객체(Object) 이해하기

클래스는 설계도이고, 객체는 설계도를 바탕으로 만들어진 결과물입니다. 간단한 클래스를 직접 만들어보면, 기계학습 모델이 어떻게 동작하는지 쉽게 이해할 수 있습니다.

# 1. 간단한 클래스(설계도) 직접 정의하기
class AIBot:
    def study(self):
        print("데이터를 열심히 학습합니다!")

# 2. 설계도를 바탕으로 객체(bot) 생성하기
bot = AIBot()

# 3. 점(.)을 찍어서 객체의 기능(메서드) 사용하기
bot.study()  # 출력: 데이터를 열심히 학습합니다!

print(df.isna().sum())

# 결측치가 포함된 행을 모두 제거하고 깨끗한 데이터를 df3에 저장합니다.

Part 1. 변수, 자료형, 입출력

프로그래밍의 가장 기본 단위인 데이터를 저장하고 화면에 나타내는 방법을 배웁니다.

1.1 변수와 자료형

변수는 데이터를 저장하는 공간이고, 자료형은 그 공간에 어떤 종류의 데이터를 저장할지 알려줍니다.

// 기본 자료형 예시
int age = 20; // 정수형 변수
float height = 175.5; // 실수형 변수
char initial = 'K'; // 문자형 변수

1.2 입출력 함수

`printf` 함수로 화면에 데이터를 출력하고, `scanf` 함수로 사용자로부터 데이터를 입력받습니다.

#include <stdio.h>

int main(void) {
  int number;
  printf("숫자를 입력하세요: ");
  scanf("%d", &number); // 사용자로부터 정수 입력받기
  printf("입력한 숫자: %d\n", number); // 입력받은 숫자 출력하기
  return 0;
}

Part 2. 연산자와 제어문

데이터를 가공하고 프로그램의 흐름을 제어하는 방법을 배웁니다.

2.1 연산자

산술 연산자(+, -, *, /), 비교 연산자(==, !=, >), 논리 연산자(&&, ||) 등을 사용하여 데이터를 가공합니다.

int x = 10, y = 5;
int sum = x + y; // 15
int is_greater = x > y; // 참(1)

2.2 제어문

`if-else`로 조건을 검사하고, `for`와 `while`로 특정 코드를 반복 실행합니다.

// 1부터 5까지의 합 구하기
int total = 0;
for (int i = 1; i <= 5; i++) {
  total += i;
}
printf("총합: %d\n", total); // 15 출력

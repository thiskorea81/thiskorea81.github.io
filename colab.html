대화형 Colab 가이드

시작하기 코드 셀 체험 텍스트 셀 꾸미기 파일 관리

Google Colab 첫걸음

웹 브라우저에서 바로 Python 코드를 작성하고 실행할 수 있는 Google Colab의 세계에 오신 것을 환영합니다. 이 가이드를 통해 코랩의 핵심 기능을 쉽고 재미있게 배워보세요.

1단계: Colab 접속 및 인터페이스 둘러보기

Colab은 '셀(Cell)'이라는 블록으로 구성됩니다. 코드를 작성하는 '코드 셀'과 설명을 위한 '텍스트 셀'이 있습니다. 아래는 Colab의 기본적인 인터페이스 모습입니다.

File.ipynb
[ ]
# 이곳이 코드를 작성하는 '코드 셀'입니다.
▼ +코드     ▲ +텍스트

이곳은 설명을 작성하는 '텍스트 셀'입니다.

접속 방법: 웹 브라우저에서 colab.research.google.com에 접속하여 '새 노트북'을 클릭하면 위와 같은 환경이 나타납니다.

2단계: 코드 셀 체험하기

코드 셀에 Python 코드를 입력하고 실행 버튼(▶)을 눌러 결과를 확인해 보세요. 아래의 코드 셀은 실제 Colab처럼 동작하는 시뮬레이션입니다. 

print("Hello, Colab!")

팁: 실제 Colab 환경에서는 `Shift + Enter` 키를 눌러 코드를 더 빠르게 실행할 수 있습니다.

3단계: 텍스트 셀 꾸미기 (마크다운)

텍스트 셀에서는 마크다운 문법을 사용해 글을 꾸밀 수 있습니다. 왼쪽 편집기에 마크다운을 입력하면 오른쪽에서 실시간으로 결과가 반영됩니다.

4단계: 파일 관리하기

Colab은 내 컴퓨터의 파일을 업로드하거나, 작업한 파일을 다운로드할 수 있습니다. 또한 Google Drive와 연동하여 파일을 편리하게 관리할 수 있습니다.

📤

파일 업로드

왼쪽 사이드바의 파일 탭(📁)에서 '세션 저장소에 업로드' 아이콘을 클릭하여 내 컴퓨터의 파일을 올립니다.

📥

파일 다운로드

업로드된 파일 목록에서 원하는 파일의 점 세 개(...) 메뉴를 클릭하여 '다운로드'를 선택합니다.

🔗

Google Drive 연동

파일 탭 상단의 '드라이브 마운트' 아이콘을 클릭하면 내 Google Drive에 바로 접근하여 파일을 사용하고 저장할 수 있습니다.

이제 Colab의 기본 사용법을 익히셨습니다. 다양한 데이터 분석과 머신러닝 프로젝트에 도전해보세요!
